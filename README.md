## SYNOPSIS
Add/Remove AD Users from an AD Group.
## DESCRIPTION
Add or remove Active Directory Users from an Active Directory Group.
## EXAMPLE
```powershell
.\Manage-ADGroup.ps1 -ADGroupName <NameOfADGroup>
```
